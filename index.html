Sistema Web Desenvolvido

2023
Sistema de Acervo Bibliográfico – Projeto Integrador (Web)
Sistema web desenvolvido como projeto acadêmico com o objetivo de otimizar a gestão de acervos em bibliotecas. A aplicação oferece suporte completo às atividades da biblioteca, como controle de entrada e saída de livros, registro de usuários e emissão de relatórios gerenciais. Voltado para alunos, professores e instituições, o sistema visa facilitar a administração do acervo e o acesso às informações.
Funcionalidades principais:
  • Cadastro, edição e exclusão de livros e categorias
  • Controle de empréstimos e devoluções
  • Autenticação de usuários
  • Pesquisa por título, categoria e instituição
  • Relatórios para suporte à gestão
Entregas:
  • Análise e levantamento de requisitos
  • Modelagem de dados
  • Protótipos das interfaces
  • Desenvolvimento completo do sistema web
  • Documentação técnica e manual do sistema
  • Apresentação final do projeto
